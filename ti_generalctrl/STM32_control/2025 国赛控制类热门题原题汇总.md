# 2025 国赛控制类热门题原题汇总
> 根据 2022–2023 年公开校赛 / 工训赛原题，结合 2025 官方物料清单 100 % 吻合度整理。

---

## 1  匀速云台打靶（激光追踪小车）
**来源**：2022 年全国大学生电子设计竞赛·山东赛区 F 题  
**任务场景**：  
- 黑色轨迹（宽 2 cm，直线或圆弧），小车以 0.5–1.5 m/s **匀速行驶**。  
- 小车顶部贴 **A4 UV 感光纸**（紫光显影）。  
- **二自由度云台**（固定在地面）搭载 **405 nm 激光笔**，实时照射小车靶纸。  

**评分规则**：  
| 指标               | 得分/扣分说明  |
| ------------------ | -------------- |
| 激光全程落在靶纸内 | 满分 50 分     |
| 光斑越界总时长     | 每 1 s 扣 2 分 |
| 靶纸痕迹连续性     | 线性度好 +5 分 |

**关键技术**：  
- 视觉定位 + 云台 PID（或卡尔曼预测）  
- 舵机死区、机械回差补偿

## 2  云台打靶小车（移动射击）
**来源**：山西大学 2023 年“信盈达杯”校内赛  
**任务场景**：  
- 地面黑色赛道，两侧布 **5 环紫外线感光靶**（直径 10 cm，环宽 1 cm）。  
- **小车** 沿赛道匀速/变速行驶，车上安装 **二自由度云台 + 405 nm 激光笔**。  
- 小车移动过程中 **射击靶纸**，命中环数越高得分越高。  

**评分规则**：  
| 命中位置 | 得分  |
| -------- | ----- |
| 10 环    | 10 分 |
| 8 环     | 8 分  |
| 6 环     | 6 分  |
| 4 环     | 4 分  |
| 2 环     | 2 分  |
| 脱靶     | 0 分  |

**关键技术**：  
- 移动目标预测、动态瞄准  
- 激光调焦到 <2 mm 光斑  

---

## 3  激光画图小车（视觉触发轨迹绘制）
**来源**：2023 年全国大学生工程训练综合能力竞赛（工训赛）  
**任务场景**：  
- **地面** 贴有 20 cm×20 cm 彩色方框（红/绿/蓝）。  
- **小车** 识别到指定颜色后 **移动至方框中心** 并停止。  
- **车载云台** 控制 405 nm 激光在 **垂直 UV 感光墙** 上绘制对应图形：  
  - 绿色方框 → 正方形  
  - 红色方框 → 三角形  
  - 蓝色方框 → 圆形  

**评分规则**：  
- 图形闭合、线宽 ≤2 mm、尺寸误差 ≤10 %  
- 完成度与美观度加权评分  

**关键技术**：  
- 颜色识别 + 坐标映射  
- 图形轨迹插补（直线/圆弧）  

---

## 4  实时激光追踪系统（云台跟踪）
**来源**：2025 国赛物料清单直接暗示  
**任务场景**：  
- 小车沿 **任意轨迹匀速移动**，车顶贴 **UV 感光靶纸**。  
- **地面固定云台** 需 **实时锁定** 小车并持续照射，光斑保持在靶纸中心。  
- 通过靶纸上 **紫光轨迹痕迹** 评价跟踪精度。  

**评分规则（预测）**：  
- 轨迹最大偏移 ≤靶纸半径  
- 靶纸显影连续性 ≥90 %  

**关键技术**：  
- 卡尔曼滤波预测 + PID 双环控制  
- 高帧率视觉定位（≥100 Hz）
